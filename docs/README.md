## 文档维护规则

1. **前台文档只放面向外部的** — 内部工作档案在 `internal/`
2. **面向投资人的** — 高质量叙事，去 AI 味，人类审阅
3. **面向用户/开发者的** — 清晰准确，可复现
4. **新增 docs/ 文件** — 必须更新本文档索引
