# Universal Collector

> **让你的收藏，Agent 能用。**

---

## 问题

你每天都在收藏——微信文章、技术博客、行业动态。但 **95% 的收藏再也不会被打开**。

不是因为懒。是因为人的消化速度远低于信息流入速度，而现有工具只解决了"怎么收"，不解决"收了之后怎么办"。

收藏变成了坟墓。

## 解决方案

Universal Collector 不改变你的习惯——你还是粘贴链接。

但收藏之后发生的事变了：**AI 自动把内容变成 Agent 能读懂、能查询、能复用的知识资产。**

你只选择信息来源。剩下的——抓取、分类、摘要、存储、检索、复用——全部交给 Agent。

## 用户

每天使用 AI Agent 的知识工作者——研究者、开发者、投资人、独立创作者。

他们的共同特征：信息 diet 高杂，日均收藏 5-20 条，但几乎不回看。

## 产品形态

```
你粘贴链接 → AI 自动分类+摘要+结构化 → Agent 可查询/复用/生成报告/转化为 Skill
```

- **Local-First**：数据存本地，Markdown + JSON，零锁定
- **Agent-Native**：数据格式和接口为 Agent 消费设计，不是为人类浏览设计
- **MCP Server**：任何 Agent 都能把你的知识库当长期记忆层

## 市场

左上象限空白——**低收藏摩擦 × 高 Agent 集成度** 的产品，市场上没有。

- 左下（低摩擦/低Agent）：Readwise、Cubox、Karakeep —— 体验好，但 Agent 无法消费
- 右上（高摩擦/高Agent）：Mem0、Supermemory —— 能力强，但只有开发者能用
- **左上：空白** —— 这就是 Universal Collector

## 商业模式

Open-Core 三步走：

1. **开源核心**（免费）→ 建立社区和信任
2. **Skill 分发**（轻量收入）→ ClawHub/WorkBuddy 生态
3. **云服务增值**（$5-8/月）→ 可选同步 + 高级 Agent 功能

盈亏平衡点：~60 付费用户，约 10-12 个月。
